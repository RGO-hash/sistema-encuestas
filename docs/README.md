# GitHub Pages - Sistema de Votación

Esta carpeta contiene los archivos para el sitio web estático de GitHub Pages.

## Contenido

- `index.html` - Página principal del sitio
- Instrucciones para ejecutar la aplicación
- Enlaces al repositorio y documentación

## Acceso

El sitio está disponible en:
https://rgo-hash.github.io/sistema-encuestas/

## Actualizar el Sitio

Para actualizar el contenido:

1. Modifica los archivos en la carpeta `docs/`
2. Commit y push a GitHub
3. El sitio se actualiza automáticamente
